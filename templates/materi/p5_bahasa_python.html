Bahasa Python

Sejarah Python

Python adalah bahasa pemrograman tingkat tinggi dan interpretatif yang banyak digunakan untuk tujuan umum. Dan yang kita tahu python sebagai nama dari ular, tetapi nama bahasa pemrograman Python berasal dari sebuah acara komedi televisi BBC lama yang disebut Monty Python's Flying Circus. Pada puncak kesuksesannya, tim Monty Python tampil dengan sketsa komedinya di depan penonton langsung di seluruh dunia, termasuk di Hollywood Bowl. Pencipta Python Guido van Rossummenamai dan bahasa tersebut dinamai python sebagai penghormatan terhadap acara TV tersebut.

Belajar Bahasa pemrograman berarti mempelajari suatu Bahasa komputer,aturan penulisan, instruksi- instruksinya, tata cara pengoprasian compiler/interpreter nya, dan menuliskan program dengan Bahasa komputer tersebut. Saat ini ada banyak bahasa pemrograman yang sudah dikembangkan seperti C, C++, Java , Python, Javascript, dan sebagainya. Salah satu bahasa pemrograman yang mudah dipelaajri adalah python. Python memiliki struktur penulisan dan istilah-istilah yang dekat dengan bahasa manusia. Bahasa pemrograman python memeiliki kemudahan karena penulisan kode program lebih sederhana sehingga lebih mudah dipelajari dan dibaca manusia daripada bahasa pemrograman lain, bahasa python memiliki library yang bisa digunakan unutk menyelesaikan berbagai maslaah seperti data since , artificial Intelligent, matematika, dll. Bahasa pyhton memiliki beberapa kelebihan antara lain :

  • Mudah dipelajari - waktu yang dibutuhkan untuk mempelajari Python lebih singkat dibandingkan dengan banyak bahasa lainnya; ini berarti bahwa memulai pemrograman sebenarnya dapat lebih cepat.
  • Mudah diajarkan - beban mengajar lebih kecil dibandingkan dengan bahasa lain; ini berarti bahwa guru dapat menekankan lebih banyak pada teknik pemrograman umum (yang independen dari bahasa), tidak membuang energi pada trik eksotis, pengecualian aneh, dan aturan yang sulit dipahami
  • Mudah digunakan untuk menulis perangkat lunak baru - seringkali mungkin untuk menulis kode lebih cepat saat menggunakan Python.
  • Mudah dipahami - juga seringkali lebih mudah memahami kode orang lain lebih cepat jika ditulis dalam Python.
  • Mudah diperoleh, diinstal, dan didistribusikan - Python gratis, terbuka, dan multiplatform; tidak semua bahasa dapat membanggakan itu.
