BAB 4 Input Output Dasar

Pendahuluan

Pada bab ini kita akan memabhas input dan output (I/O) dasar dalam bahasa pemrograman Python, yang merupakan elemen penting dalam memungkinkan interaksi antara program dan pengguna serta lingkungan eksternal. Input merujuk pada data yang dimasukkan ke dalam program, misalnya melalui keyboard atau berkas, sedangkan output adalah hasil yang dihasilkan oleh program dan ditampilkan kepada pengguna, biasanya melalui layar monitor atau disimpan ke dalam berkas. Pemahaman yang baik tentang bagaimana memanipulasi input dan output dalam Python memungkinkan pengembang untuk membuat program yang lebih dinamis dan interaktif, serta mengimplementasikan solusi yang lebih efektif untuk berbagai masalah pemrograman.

Tujuan Pembelajaran :
Setelah memahami input dan output dasar pada bahasa python di media pembelaajran, siswa diharapakan :
  • Peserta didik memahami kegunaan dan menerapkan fungsi input() untuk menerima inputan dari user.
  • Peserta didik mampu memodofikasi tipe data inputan dan menerapkannya dalam program.
  • Peserta didik mampu memahami dan dapat menentukan penggunaan tipe data pada pemorgraman pyhton sesuai dengan keperluan.
  • Peserta didik mampu mengetahui dan memahami operator dasar yang ada di python.
  • Peserta didik mampu menentukan dan menerapkan penggunaan operator dasar pada bahasa pyhton sesuai dengan keperluan.
