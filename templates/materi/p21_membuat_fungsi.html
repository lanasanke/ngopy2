Membuat dan Memanggil Fungsi

Di Python, sebuah fungsi didefinisikan dengan menggunakan kata kunci def diikuti nama fungsinya, kemudian diikuti dengan blok kode yang akan dijalankan ketika fungsi dipanggil, berikut sturktur dasar mendefinisiakn fungsi dalam python :

Struktur Dasar Percabangan Fungsi

def nama_fungsi :
  stetmen_yang_dijalankan

Pada gambar diatas adalah struktur dasar untuk mendefiniskan fungsi,baris pertama diawali dengan kata kunci def lalu diikuti oleh nama fungsinya, baris kedua dan seterunya adalah blok kode fungsi yang kita buat. Berikut contoh dari fungsi yang dibuat oleh user :

Contoh 5.1 Fungsi yang dibuat oleh user

def sapa() :
  print("Selamat Pagi")

Pada contoh diatas kita membuat sebuah fungsi untuk menampilkan salam. Pada baris pertama kita membuat fungsi dengan kata kunci def lalu diikuti nama fungsi nya yaitu sapa (), lalu pada blok kode fungsi salam() kita menggunakan print() untuk mencetak string “Selamat Pagi”, jalankan kode program pada IDE dibawah ini!.

Fungsi sapa() diatas belum bisa langsung digunakan untuk mencetak string,

Untuk memanggil fungsi yang sudah kita buat sebelumnya yaitu dengan mengetikan nama fungsinya, seperti contoh sebelumnya kita sudah mendefiniskan fungsi dengan nama sapa().kita harus memanggilnya dulu agar kita bisa menggunakan fungsi yang sudah kita buat sebelumnya.jalankan kode program pada IDE dibawah ini!.
