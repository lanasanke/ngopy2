Input Dasar

Pada materi sebelumnya kita sudah membuat beberapa varaibel yang nilainya diisikan secara langsung pada kode program sehingga nilai apda variabel tersebut bersifat statis, jika kita ingin mengubah nilai dari variabel trsebut maka kita harus mengeditnya pada kode program. Dalam python kita bisa melakukannya menggunakan fungsi bawaan python yaitu input(). Fungsi ini akan menampilkan promp di layer yang meminta pengguna untuk mengisikan nilai. Berikut contoh penggunaan fungsi input () :

Contoh kode 4.1

input("Masukan nilai hewan =")

Secara default inputan yang diterima dari fungsi input() bertipe string, sehingga nilai tersebut tidak bisa digunakan untuk melakukan operasi tertentu seperti aritmatika dan lain-lain. Perhatikan kode program dibawah ini yang digunakan untuk mencari nilai pangkat dua dari sebuah angka :

Contoh kode 4.2

angka = input("Masukan angka =")
hasil = angka**2
print(hasil)

Python akan memberikan pesan error karena python membaca inputan yang diberikan sebagai string sehingga tidak bisa dilakukan operasi perpangkatan. Agar pyhon dapat membaca inputan yang diberikan dari fungsi input() sesuai dengan tipe data yang kita inginkan maka kita harus melakukan konversi tipe data pada fungsi input dengan cara menambahkan tipe data yang kita inginkan pada fungsi input(). Berikut contoh untuk mengkonversi tipe data pada fungsi input () :

Contoh kode 4.3

#fungsi input() membaca inputan sebagai integer
bilangan_bulat = int(input("Masukan bilangan bulat = "))

#fungsi input() membaca inputan sebagai float
bilangan_pecahan = float(input("Masukan bilangan pecahan = "))

print("varaibel bilangan_bulat bertipe",type(bilangan_bulat))
print("varaibel bilangan_pecahan bertipe",type(bilangan_pecahan))

Menambhakan tipe data sesuai dengan jenis data yang kita inputakn pada fungsi input() akan membuat python mengenali niali yang diinputkan berdasarkan tipe data yang didefiniskan.
