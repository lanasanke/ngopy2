Percabangan Bersarang

Percabangan if..else bersarang adalah struktur kendali percabangan dimana ada penggunaan beberapa blok if..else dalam percabangan if..else yang lain. Penggunaan percabangan ini memungkinkan kita untuk memeriksa kondisi yang lebih kompleks dan beruntun. Setiap blok percabangan dijalankan berdasarkan apakah kondisi yang terkait dengan blok tersebut benar atau salah. Perhatikan flowcahr dibawah ini agar lebih memahami bagaimana cara kerja percabangan bersarang.

Pada flowchart diatas memiliki 3 kondisi dimana jika kondisi1 bernilai benar maka program akan masuk ke kondisi 2 dan memriksa sesuai dengan kondisi yang ditentukan. Sedangkan jika kondisi1 bernilai salah maka program akan masuk ke kondisi 3 dan memeriksa sesuai dengan kondisi yang ditentukan. jika dituliskan maka flowchart diatas akan menjadi seperti berikut :

Struktur Dasar Percabangan Bersarang

if kondisi1 :
   if kondisi2:
     Stetment_jiika_kondisi2_true
   else_kondisi2:
      Stetment_jiika_kondisi2_false
else_kondisi1:
  Stetment_jika_kondisi1_false
   ……

Beberapa hal yang harus dipertimbangkan saat menggunakan if bersarang adalah letak else dari setiap blok if yang berbeda, selalu perhatikan penempatan else dari setiap blok if yang merupakan pasangannya. Selain itu perhatikan juga indentasi saat menggunakan if bersarang karena pada pyhton sebuah blok kode akan dikelompokan dari jumlah indentasinya. Berikut contoh program percabangan bersarang :

Struktur Dasar Percabangan Bersarang

nilai = 85
if nilai >= 80:
  print("Baik")
else :
   if nilai >= 60 :
     print("Cukup")
   else :
     print("Kurang")

Contoh program diatas adalah program untuk menentukan predikat dari sebuah nilai dengan kondisi jika nilai >= 80 maka akn mencetak kalimat “Baik”, namun jika nilai < 80 maka akan masuk ke percabangan kedua. Pada percabangan kedua nilai akan di periksa kembali apakah nilai tersebut >= 60 jika iya maka akan mencetak kaliamt “Cukup”, namun jika tidak akan mencetak kalimat “Kurang”.
