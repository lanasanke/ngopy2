Operator Penugasan

Operator penugasan atau assignment operator pada Python adalah operator yang digunakan untuk mengisi nilai ke dalam variabel. Operator ini memungkinkan programmer untuk memberikan nilai ke variabel dan kemudian menggunakan nilai tersebut dalam operasi lain pada program. Berikut ini adalah beberapa operator penugasan yang digunakan pada Python:

Nama Operator Simbol Keterangan Contoh
Penugasan = Digunakan untuk memberikan nilai ke vriabel
Penjumlahan += Digunakan untuk menjumlahkan suatu nilai dengan nilai variabel
Pengurangan -= Digunakan untuk mengurangkan suatu nilai dengan nilai variabel
Perkalian *= Digunakan untuk mengalikan suatu nilai dengan nilai variabel
Pembagian /= Digunakan untuk membagi suatu nilai dengan nilai variabel
Modulo %= Digunakan Untuk mengambil sisa pembagian dari varaiabel
Perpangkatan **= Digunakan untuk memangkatkan nilai varaiabel dengan suatu nilai
Pembagian Bulat //=

Digunakan untuk membagi nilai varaibel dengan suatu nilai dan
mengembalikan dalam bentuk bilangan bulat
