Cara Kerja Fungsi

Perhatikan gambar diabwah ini !

Pada gambar diiatas adalah gambaran bagaimana sebuah fungsi sapa() dipanggil dan dijalankan. saat fungsi sapa() dipanggil, maka python akan mengingat dimana bagian fungsi dipanggil dan track back ke dalam fungsi yang bersangkutan. Fungsi sapa() akan mengeksekusi blok kode fungsi, yaitu mencetak string "Selamat Pagi". Setelah blok kode dieksekusi output akan dikembalikan ke tempat dimana fungsi sapa() dipanggil.

Ada dua hal yang harus diperhatikan saat memanggil fungsi :

  • Jangan memanggil fungsi yang belum didefinisikan sebelumnya Python membaca kode dari atas kebawah. Python tidak akan mencari fungsi yang belum didefiniskan sebelumnya. Artinya kita harus mendefinisikan fungsi terlebih dahulu sebelum memanggilnya. Perhatikan potongan kode berikut : Pada potngan kode diatas kita memindah fungsi sapa() menjadi di akhir baris, jika kode yang dituliskan seperti ini maka kode akan menghasilkan error karena fungsi sapa() dipanggil terlebih dahulu sebelum didefinisikan.

  • Jangan Mendefinisikan nama fungsi sama dengan nama vaiabel, Perhatikan potongan kode program brikut : Pada potongan kode diatas kita mengisi value ke dalam variabel sapa akan membuat python bingung dan menganggap sapa adalah variabel dan melupakan fungsi yang bernama sapa().
