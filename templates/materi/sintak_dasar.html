Sintak Dasar Python

Setiap bahasa memiliki aturan-aturan penulisannya, tidak terkecuali bahasa pemrograman. Setiap bahasa pemrogramana memeiliki aturan-aturannya sendiri. bahasa python memiliki beberapa aturan yang sedikit berbeda dari bahasa pemrograman lain, namun secara umum aturan penulisannya sama dengan bahasa pemrograman lain.

Bersifat Case Sensitive

Beberapa bahasa pemrograman menggunakan kosep case sensitive, termasuk bahasa python. Case sesistive adalah saat kita menuliskan kode menggunakan huruf kapital dan huruf kecil akan memiliki arti yang berbeda walaupun dengan kalimat yang sama. Penggunaan kosep case sensitive akan mengurangi kesalahan dalam menuliskan kode dan membuat kita konsisten dalam menuliskan kode. Selalu perhatikan dan teliti dalam menuliskan kode program. perahtikan contoh berikut :

Contoh kode 2.1

Tanpa Semicolon

Beberpa bahasa pemrograman seperti C++, PHP, Java, dll menggunakan simbol semi colon atau titik koma ( ; ) untuk mengakhiri sebuah stetment atau baris kode. Tetapi pada bahasa python penggunaan simbol semicolon untuk mengakhiri sebuah baris stetment adalah sebuah pilihan.Pyhton membaca baris baru (/n) sebagai akhir dari sebuah stetmen. Setiap kali ada perubahan baris, Python mengasumsikan bahwa pernyataan sebelumnya telah berakhir. Kita dapat menulis setiap pernyataan pada baris yang berbeda tanpa perlu menggunakan semicolon untuk memisahkannya. Perhatikan kode program dibawah ini

Conoth kode 2.3

print("Selamat Pagi")
print("Pagi ini cuaca cerah")

Pada gambar diatas kita memiliki dua stetmen pertama pada baris pertama menampilkan kalimat “ Selamat Pagi “. Stetmen kedua pada baris kedua menampilkan kalimat “ Pagi ini cuaca cerah”. Namun kita juga bisa membuat kedua stetmen ini dalam satu baris menggunakan simbol semicolon. Perhatikan conoth program dibawah ini!.

Contoh kode 2.4

print("Selamat Pagi"); print("Pagi ini cuaca cerah")

Pada contoh program diatas pada dasarnya sama saja dengan contoh program sebelumnya, tetapi contoh yang ini menggunakan simbol semicolon atau titik koma sebagai akhir dari stetment. Silahkan coba contoh program pada IDE berikut :
