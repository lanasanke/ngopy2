Variabel

Dalam matematika variable adalah “wadah “ untuk menyimpan suatu nilai. Dalam kehidupan sehari-hari variabel diibaratkan seperti sebuah baskom yang bisa digunakan untuk menampung makanan, air, sayuran dan lain-lain seperti contoh dibawah ini.

Dari gambar diatas kita bisa menganalogikan variabel sebagai sebuah baskom dan sayuran sebagai nilai yang ditampung oleh varaiabel baskom. Sehingga jika sudah ditulis dalam bentuk kode maka varaibel baskom yang berisi nilai “Sayuran” akaan menajdi sebagai gambar berikut :

Dalam pemrograman variable memiliki kegunaan yang sama yaitu untuk menyimpan suatu nilai agar bisa diolah oleh komputer. Dalam bahasa python memiliki aturan-aturan untuk mendefinisikan variable. Berikut aturan-aturan mendefinisiakn variable :

  • Nama variabel harus dimulai dengan huruf atau garis bawah (_).
  • Nama variabel tidak boleh dimulai dengan angka.
  • Nama variabel hanya dapat mengandung karakter alfanumerik dan garis bawah.
  • Nama variabel bersifat case sensitive, sehingga variabel "nama" dan "Nama" dianggap berbeda.
  • Hindari menggunakan kata kunci (keyword) Python seperti "if", "else", "while",dan sebagainya sebagai nama variabel.
  • Usahakan memberikan nama variabel yang deskriptif, sehingga mudah dipahami dan dibaca.

Berikut adalah Contoh penulisan variabel yang benar :

umur = 20
nama_lengkap = "Muhammad Maulana"
jumlahsiswa = 34

Berikut adalah contoh penulisan variabel yang salah:

1umur = 27 # tidak boleh dimulai dengan angka
nama lengkap = "John Doe" # Tidak boleh menggunakan Spasi

if = 50 # menggunakan kata kunci Python sebagai nama variabel
