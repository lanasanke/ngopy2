Operator

Operator adalah simbol yang merepresentasikan sebuah operasi pada satu atau beberapa operand. Operator juga bisa berfungsi untuk memanipulasi nilai suatu variable. Variable yang nilainya dimodifikasi oleh operator disebut operand. Operator dalam bahasa python dikelasifikasikan menajdi beberapa kelompok .

Operand adalah sebuah nilai atau variabel yang akan kita lakukan lakukan operasi seperti perhitungan artimatika, perbandingan, mengisi nilai dan lain-lain.

    Operator Artimatika

    Operator aritmatika adalah operator yang digunakan untuk menyelesaikan operasi matematika. Berikut daftar operator yang termasuk dalam operator aritmatika

    Nama Operator Simbol Keterangan Contoh
    Penjumlahan + Digunakan untuk menjumlahkan suatu nilai atau nilai variabel
    Pengurangan - Digunakan untuk mengurangkan suatu nilai atau nilai variabel
    Perkalian * Digunakan untuk mengalikan suatu nilai atau nilai variabel
    Pembagian / Digunakan untuk membagi suatu nilai atau nilai variabel
    Modulo % Digunakan Untuk mengambil sisa pembagian bilangan atau nilai varaiabel
    Perpangkatan ** Digunakan untuk memangkatkan nilai varaiabel dengan suatu nilai
    Pembagian Bulat //

    Digunakan untuk membagi suatu bilangan atau nilai varaibel dengan suatu bilangan dan
    mengembalikan hasil operasi dalam bentuk bilangan bulat
