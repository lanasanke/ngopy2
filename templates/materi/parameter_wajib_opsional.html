Parameter Wajib dan Opsional

Parameter wajib adalah parameter yang harus diberikan Ketika memanggil fungsi, jika kita tidak memberikan argument untuk parameter wajib maka pyhon akan memberikan pesan error atau exeption. Parameter wajib disebut juga dengan parameter posisi karena diidentifikasi berdasarkan posisinya di fungsi. Perhatikan contoh berikut :

Pyhton akan mengembalikan 3 argumen yang diberikan untuk 3 parameter sehingga fungsi berjalan dengan baik. Jika contoh program sebelumnya dimodifikasi sehingga menjadi berikut :

Python akan memberikan output pesan error karena jumlaha rgumen yang diberikan tidak sesuai dengan jumlah parameter yang didefinisikan. Fungsi siswa() memiliki 3 parameter yaitu nama, jenis_kelamin dan kelas. Namun argumen yang diberikan hanya 2 yaitu argumen “maulana” untuk parameter nama dan argument “laki-laki” untuk jenis_kelamin.
